## 技术选型讨论
- Master 确认使用 **OpenClaw + Claude Code** 模式开发
- 讨论了前端架构选择：UniApp vs React Native vs Flutter vs Kotlin 原生
- **结论：UniApp (Vue 3) 为首选方案**
  - 理由：H5 模式下 OpenClaw 可用 browser 工具自测（覆盖~80%功能）
  - 云打包直出 Android APK，无需 Android Studio
  - Vue 语法团队学习成本最低
  - 还能编译 vivo 快应用格式（比赛加分）
- H5 vs Android 差异分析：布局/逻辑/API调用一致（80%），拍照体验/LBS精度/推送通知/权限弹窗需真机测（20%）
- 工作流：我负责写代码+H5测试，队友负责真机验证
- iOS 暂不考虑（比赛只要 Android）
