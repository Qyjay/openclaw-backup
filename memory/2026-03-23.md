## MiniMax 图像生成 API 发现
- **正确端点**：`POST https://api.minimax.chat/v1/image_generation`
- **正确模型名**：`image-01`（不是 `nano-banana`）
- API Key 用 `MINIMAX_TOKENPLAN_API_KEY`
- NanoBanana 代理地址 `api.minimax.io` 的 `/v1/gemini` 路径全部 404，不可用
- `api.minimax.chat` 才是正确的图像 API 域名
