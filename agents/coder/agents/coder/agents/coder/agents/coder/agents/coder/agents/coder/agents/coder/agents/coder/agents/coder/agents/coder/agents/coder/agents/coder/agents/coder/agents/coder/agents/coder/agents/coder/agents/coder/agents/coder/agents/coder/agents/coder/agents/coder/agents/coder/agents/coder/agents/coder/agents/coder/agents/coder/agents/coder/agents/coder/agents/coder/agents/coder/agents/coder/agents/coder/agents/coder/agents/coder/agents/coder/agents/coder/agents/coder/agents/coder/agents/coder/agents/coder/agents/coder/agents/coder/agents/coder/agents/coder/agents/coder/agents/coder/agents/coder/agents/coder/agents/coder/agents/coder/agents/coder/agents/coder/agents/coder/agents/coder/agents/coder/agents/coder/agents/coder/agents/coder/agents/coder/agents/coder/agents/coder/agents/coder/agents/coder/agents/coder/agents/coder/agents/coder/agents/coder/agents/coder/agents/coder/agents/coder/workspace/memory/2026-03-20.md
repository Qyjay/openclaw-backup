## 技术备忘
- Nano Banana API：`https://api.minimax.io/v1/gemini/v1beta/models/g3-pro-image-preview:generateContent`
- 需要 headers: `Authorization: Bearer <GEMINI_API_KEY>`, `X-Biz-Id: op`
- API Key 和配置在 `.env` 文件中
- 生成图片时 timeout 至少 180s，参考图上传时更慢需要 240s
- `source .env` 不会 export 变量，需要用 `export $(grep -v '^#' .env | xargs)`
