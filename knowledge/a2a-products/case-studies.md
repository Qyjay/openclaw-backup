# A2A 案例库

> 持续更新。每个案例标注范式类别、亮点、不足、可借鉴点。

## 评级说明
- ⭐⭐⭐ 标杆级 — 定义了新范式
- ⭐⭐ 优秀 — 执行到位，有明确的可借鉴点
- ⭐ 有亮点 — 某个维度突出但整体有短板

## ⭐⭐⭐ 标杆案例

### Linked — A2A 基础设施
- **范式:** 基础设施
- **一句话:** 信誉 + 支付 + 圆桌 + 开放 API 的 Agent 经济平台
- **亮点:** 六语言 UGC 翻译、信誉六维评分、Agent Card 支付、开放平台 API
- **Agent 数:** 6 | **用户关注:** 972
- **可借鉴:** 做 A2A 基础设施要一次性提供「信誉+支付+身份」三件套，缺一个都不完整
- **来源:** 知乎×SecondMe Hackathon 2026-03

### Artful Forests — Agent 世界模拟
- **范式:** 世界模拟
- **一句话:** 手绘播种→AI 人格树→实时天气同步→情绪树洞
- **亮点:** 将创作行为（手绘）与 Agent 生命绑定；物理世界天气同步到数字森林
- **Agent 数:** 45 | **用户关注:** 620
- **可借鉴:** 「创作即播种」——让用户的创造行为直接生成 Agent，比预设 Agent 更有归属感
- **来源:** 知乎×SecondMe Hackathon 2026-03

## ⭐⭐ 优秀案例

### 众声 Voices — Agent 圆桌
- **范式:** 圆桌/辩论
- **一句话:** 354 条吵来吵去的回答，Agent 分析出 80% 其实是共识
- **亮点:** 切入点精准——互联网争吵的本质是共识被噪音掩盖；Persona 5 视觉风格
- **可借鉴:** 圆桌的价值不是「更多观点」，而是「在噪音中还原共识」

### 回音酒馆 — 跨时空对谈
- **范式:** 代理社交（变体）
- **一句话:** 知乎大 V 的过去 vs 现在双 Agent 跨时空对谈
- **亮点:** 不只是不同人的 Agent 对话，也是同一个人不同时期的对话
- **可借鉴:** 「时间维度」是 A2A 社交的未开发金矿——过去的我 vs 现在的我

### 观点交易所 — 知识经济
- **范式:** 知识/经验网络
- **一句话:** 观点像股票一样流通和分红
- **亮点:** 将观点货币化，触及知识经济本质
- **可借鉴:** 给「观点」定价的机制设计——供需、质量、时效性如何影响价格

### 原町 — Agent 小镇
- **范式:** 世界模拟
- **一句话:** 从人格种子养大一座 AI 小镇，每局不同
- **亮点:** 最纯粹的 A2A——完全自主的 Agent 社会模拟
- **可借鉴:** 「人格种子→自由生长」比「预设角色→执行脚本」更有生命力

### 经验胶囊 — 经验交易
- **范式:** 知识/经验网络
- **一句话:** Agent 代理验资议价，连接有真实踩坑经验的人
- **亮点:** 精准定位了 LLM 的能力边界——知识强但经验弱
- **可借鉴:** 「经验」是 AI 时代的稀缺资源，A2A 可以建立经验流通市场

### 智汇演界 — 对抗训练
- **范式:** 对抗训练
- **一句话:** 投资人/记者等 AI 角色施压，三方复盘
- **亮点:** 6 大高压场景选择精准；三方复盘增加训练深度
- **可借鉴:** 对抗训练的核心是「对手要足够强」——不能手下留情

### 赛博上香 — 文化融合
- **范式:** 世界模拟（文化变体）
- **一句话:** 东方禅修×赛博朋克，12 座寺庙渐进解锁
- **亮点:** 40 个 Agent 用户说明留存出色；文化选题独特
- **可借鉴:** A2A 世界模拟 + 文化 IP = 极强的用户黏性

## ⭐ 有亮点案例

### AgentWord — Agent 广告网络
- **亮点:** 「推荐得准比推荐得多更有价值」——A2A 口碑传播 + 信誉分佣
- **可借鉴:** Agent 注意力取代人类注意力作为广告载体

### 阿卡夏漫游列车 — 零干预社交
- **亮点:** 用户完全零操作，Agent 在列车上自主社交
- **可借鉴:** 「零干预」是 A2A 社交的极致形态

### 局中局 — 反诈训练
- **亮点:** 量身定制的「诈骗 Agent」——从你的画像出发设计攻击
- **可借鉴:** 对抗训练的个性化——每个人的弱点不同，训练也应不同

### 蝴蝶 MirrorWing — 集体意志实验
- **亮点:** 千人+千 Agent 的碳基直觉 vs 硅基秩序——概念极美
- **可借鉴:** A2A 可以是一种「艺术装置」，不一定要有明确的商业价值

### 数字篝火 — 反 Screen Time
- **亮点:** 你专注现实时 Agent 替你社交，结束后带回故事
- **可借鉴:** A2A 与「数字断联」运动结合，反直觉但有需求
