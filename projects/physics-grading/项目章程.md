## 项目愿景

为高中物理教师提供一套 AI 智能批改系统，自动识别学生作业中的**文字、公式、物理示意图**，完成逐步推导检查与评分反馈——将单份作业批改时间从 3-5 分钟缩短至 15 秒内，实现即时、一致、个性化的过程性评价。

## 商业目标

1. 文字 OCR 识别准确率 ≥ 90%，公式识别准确率 ≥ 85%
2. 受力图关键元素识别率 ≥ 80%，批改与人工一致率 ≥ 80%
3. 单题平均批改耗时 ≤ 15 秒，系统正常运行率 ≥ 95%

## 范围

| ✅ In Scope | ❌ Out of Scope |
|------------|----------------|
| 手机拍照 / 扫描件图像输入 | 实时视频流批改 |
| 文字、公式、受力分析图识别 | 化学 / 数学等其他学科 |
| RAG + 大模型逐步批改、评分与反馈 | 本地大模型部署（采用 API） |
| 智能体多 Agent 协作批改 | 学生端 / 教师端完整管理平台 |

## 团队工作规范

**任务完成标准（DoD）：** 代码通过 Code Review + 单元测试覆盖核心逻辑 + 接口文档同步更新 + 在开发环境可演示运行

**文档提交规范：** 所有文档存飞书知识库，命名格式「模块名-文档类型-版本号」；Sprint 结束前提交本迭代技术文档；会议纪要由轮值记录人在会后 24h 内发群

**冲突沟通流程：** 技术分歧 → 飞书群讨论 → 无法达成一致时由 PM 决策；进度延期 → 第一时间在 Daily Scrum 报阻塞 → PM 协调资源或调整优先级
