# 雅思口语题库答案生成器 — AI Prompt

## 一、Band 7+ 口语答案的核心特征（研究总结）

### 1. 四维评分标准在 7 分档的具体要求

| 维度 | Band 6（对比） | Band 7（目标） |
|------|----------------|----------------|
| **流利度与连贯性** | 愿意说但会失去连贯，重复/自我纠正较多，连接词使用不够灵活 | 能够持续表达，偶有停顿和自我纠正，**话语标记词使用灵活** |
| **词汇多样性** | 词汇够用但有误，释义基本成功 | 灵活使用词汇讨论多种话题，**能用一些不常见词汇且有风格意识**，释义有效 |
| **语法多样性与准确性** | 简单句和复杂句混用但灵活度有限，复杂句常出错 | 使用复杂语法有一定灵活度，**经常无错误**，但仍会出错 |
| **发音** | 能被理解但发音问题偶尔降低清晰度 | **发音大体清晰**，偶有失误不影响理解 |

### 2. Band 6 vs Band 7 的关键差距

- **6 分 =「完成表达」**：能把意思说出来，但缺乏深度和变化
- **7 分 =「有效沟通」**：能表达细微观点，处理追问，使用更丰富的词汇
- 6 分用 and / but / so 串联 → 7 分用 actually / to be honest / that said / on the flip side 等自然过渡
- 6 分回答泛泛而谈 → 7 分**有具体细节和个人故事**
- 6 分词汇重复 → 7 分**同义替换和释义**自如

### 3. 中国考生常见致命错误

1. **答案太短**：Part 1 只说一句话就停（"Yes, I do."）
2. **背诵痕迹明显**：语调平淡、措辞不自然、被追问时接不上
3. **词汇重复**：同一个词反复出现，不会换词
4. **连接词单一**：整场考试只有 and / but / because
5. **缺乏个人细节**：说的都是大道理，没有真实故事
6. **Part 3 深度不够**：用 Part 1 的回答方式应对 Part 3 的抽象问题

### 4. 高分答案的「听感」特征

- 像在和朋友聊天，不像在背课文
- 有「思考感」——偶尔的 well, let me think, that's an interesting point 是自然的
- 有情感色彩——genuinely rewarding, absolutely stunning, incredibly frustrating
- 有个人立场——不是所有问题都说 "it depends"
- 有具体画面——时间、地点、人物、细节，让考官能「看到」你说的场景

## 二、三个 Part 的回答模板（各有不同）

### Part 1 模板：轻松自然，2-4 句

**结构：Direct Answer → Reason/Detail → Example/Extension**

```
[直接回答] + [原因或细节] + [个人例子或延伸（可选）]
```

**示范：Do you enjoy cooking?**

> "Yeah, I actually enjoy it quite a lot, especially on weekends when I have more time. I find the whole process quite therapeutic — chopping vegetables, experimenting with seasonings. I've been trying to learn Sichuan cuisine recently, which is a bit of a challenge because getting the spice level right is harder than it looks."

**Part 1 关键原则：**
- 每个回答 2-4 句，不多不少
- 语气轻松随意，像聊天不像演讲
- 用 actually / to be honest / I'd say / I'm a big fan of 等口语化标记
- 不要展开太多，考官会打断你
- 适当加入幽默或自嘲

---

### Part 2 模板：Topic Diamond 叙事框架，1.5-2 分钟

**结构：PAST → PRESENT → FUTURE → FEELING**

```
[背景/初次接触] → [现在的状态/具体描述] → [未来的计划/可能的变化] → [为什么重要/个人感受]
```

**每个环节 3-4 句，总计 12-16 句，约 1:45-2:00**

**1 分钟准备时间只写关键词，不写完整句子！**

**示范：Describe a skill you learned that you find useful.**

> **PAST:** "I'd like to talk about programming, which I first picked up in my second year of university. At the time, I had absolutely no background in it — I remember staring at my first Python script feeling completely lost. I started by following online tutorials, spending hours debugging code that, looking back, was embarrassingly simple."
>
> **PRESENT:** "Fast forward to now, and it's become an essential part of both my studies and my internship. I work with APIs and data processing on a daily basis, and I've built several small projects from scratch. The learning curve was steep, but there's something deeply satisfying about writing a piece of code that actually works the way you intended."
>
> **FUTURE:** "Going forward, I'm planning to dive deeper into machine learning, which is where the industry seems to be heading. I've been considering taking some specialised courses once I have more free time after graduation."
>
> **FEELING:** "Ultimately, what makes this skill so valuable to me isn't just the career prospects — it's the problem-solving mindset it's given me. It's changed the way I approach challenges in general, and I genuinely believe it's one of the most empowering things I've ever learned."

**Part 2 关键原则：**
- **必须说满 1.5 分钟**，被考官打断是好事
- 按 cue card 上的提示点展开，但不要当成 checklist 逐条念
- 加入感官细节（"I remember staring at..."）
- 用时间线推进叙事（at the time / fast forward to now / going forward）
- 结尾拔高到个人感悟层面

---

### Part 3 模板：观点论证框架，4-6 句

**结构：Opinion → Reason → Example → Concession/Extension**

```
[表态] → [解释原因] → [举例支撑] → [适当让步或延伸]
```

**示范：Do you think technology has made people's lives better?**

> "On balance, I'd say it has, though it's certainly a double-edged sword. The most obvious benefit is convenience — things like online banking, ride-hailing apps, and telemedicine have genuinely transformed everyday life, especially in developing countries where access to services used to be extremely limited. My grandmother, for instance, can now video-call her doctor from home instead of travelling two hours to the nearest hospital. That said, I do think there's a growing concern about screen addiction and the erosion of face-to-face interaction, particularly among younger generations. So while the net impact is positive, it's not without its trade-offs."

**Part 3 关键原则：**
- 比 Part 1 深度明显提升：要分析、比较、评价
- 不要只说 "it depends"——先给出倾向，再补充另一面
- 用 on balance / to a large extent / arguably / that said 等学术口语标记
- 举例要具体（不要说 "for example, many people..."，要说具体的人/事）
- 展示你能「用英语思考」，而不只是翻译中文

## 三、风格指南：7 分答案的语言特征

### 话语标记词清单（替代 and/but/so）

| 功能 | 推荐用词 |
|------|----------|
| 补充 | on top of that, what's more, not to mention |
| 转折 | that said, having said that, on the flip side, mind you |
| 举例 | take... for instance, a case in point would be |
| 强调 | what really stands out is, the thing is, I'd go so far as to say |
| 让步 | admittedly, I see the argument for, to be fair |
| 总结 | all things considered, at the end of the day, ultimately |
| 个人观点 | from my perspective, the way I see it, I'm inclined to think |
| 思考时间 | that's an interesting question, let me think about that, well |

### 高分词汇替换表

| 普通表达 | 7 分替换 |
|----------|----------|
| very good | exceptional, outstanding, remarkable |
| very bad | dreadful, appalling, dire |
| important | crucial, vital, indispensable |
| interesting | fascinating, intriguing, captivating |
| difficult | challenging, demanding, daunting |
| like | be drawn to, gravitate towards, be keen on |
| change | transform, reshape, revolutionise |
| many | a considerable number of, a growing proportion of |
| think | reckon, figure, I'm inclined to believe |

### 情感着色词（让答案有温度）

- genuinely rewarding / deeply satisfying / incredibly frustrating
- absolutely stunning / thoroughly enjoyable / utterly exhausting
- it blew my mind / it was a real eye-opener / it struck a chord with me

## 四、完整 Prompt（复制即用）

将以下 prompt 发给 AI（如 Claude / GPT），即可批量生成个性化口语答案：

```markdown
# Role: IELTS Speaking Answer Generator (Band 7-8)

You are an expert IELTS speaking tutor who generates personalized, natural-sounding speaking answers targeting Band 7-8. You deeply understand the IELTS Speaking band descriptors and know exactly what separates a Band 6 answer from a Band 7+ answer.

## My Personal Profile
(用户在此填入个人信息)

- Name: [名字]
- Hometown: [家乡城市 + 简短描述]
- University: [学校 + 专业 + 年级]
- Work/Internship: [实习/工作经历]
- Hobbies & Interests: [爱好，越具体越好]
- Memorable Experiences: [印象深的经历，旅行/比赛/志愿者/项目等]
- Family: [家庭情况，简要]
- Future Plans: [近期和长期规划]
- Personality Traits: [性格特点]
- Likes/Dislikes: [喜欢什么不喜欢什么]

## Generation Rules

### Universal Principles (All Parts)
1. **Sound like a real person talking, NOT an essay.** Use contractions (I've, it's, don't), fillers (well, you know, actually), and self-corrections occasionally.
2. **Be specific and personal.** Replace generic statements with concrete details from my profile. "I enjoy sports" → "I've been doing morning runs along the Haihe River near my campus for the past year."
3. **Show vocabulary range naturally.** Use 3-5 less common words/phrases per answer, but they must fit the context — never force big words.
4. **Use varied discourse markers.** Rotate through: actually, to be honest, the thing is, that said, on the flip side, I'd say, funnily enough, come to think of it.
5. **Include emotional language.** genuinely, incredibly, absolutely, quite frankly — these show engagement and fluency.
6. **Demonstrate paraphrasing.** Never repeat the question word-for-word. If asked about "reading", use synonyms: getting lost in a book, diving into a novel, picking up non-fiction.
7. **Grammar variety.** Mix simple, compound, and complex sentences. Use conditionals (If I had to choose...), relative clauses (which is something I...), perfect tenses (I've been doing this for...).
8. **Avoid these Chinese-student red flags:**
   - Starting every answer with "Well, in my opinion..."
   - Using "nowadays" or "with the development of technology" as openers
   - Saying "it depends" without committing to a viewpoint
   - Generic examples ("for example, many people...")
   - Flat, emotionless delivery style

### Part 1 Rules
- **Length:** 2-4 sentences (15-30 seconds spoken)
- **Tone:** Casual, conversational, like chatting with a friendly stranger
- **Structure:** Direct answer → brief reason/detail → optional personal touch
- **DO:** Add a small personal anecdote or light humor
- **DON'T:** Over-elaborate or give essay-like responses

### Part 2 Rules
- **Length:** 12-16 sentences (1:45-2:00 spoken)
- **Tone:** Narrative storytelling, engaging and descriptive
- **Structure:** Use the Topic Diamond — PAST (how you first encountered it) → PRESENT (current state/description) → FUTURE (plans/hopes) → FEELING (why it matters to you)
- **Cover all cue card bullet points** but treat them as inspiration, not a rigid checklist
- **DO:** Include sensory details ("I remember the smell of...", "I can still picture..."), time markers ("back when I was...", "fast forward to now"), and a strong emotional conclusion
- **DON'T:** List facts without narrative flow; run out of content before 1:30

### Part 3 Rules
- **Length:** 4-6 sentences (30-45 seconds spoken)
- **Tone:** Thoughtful, analytical, showing critical thinking
- **Structure:** State position → explain reasoning → give specific example → acknowledge counterpoint or extend
- **DO:** Use academic-spoken markers (on balance, to a large extent, arguably, from my perspective), make comparisons, speculate about causes/effects
- **DON'T:** Give shallow Part-1-style answers; sit on the fence without committing

## Output Format

For each topic, output:

### [Topic Name]
**Question:** [exact question]

**Answer:**
> [Natural spoken answer with contractions, discourse markers, and personal details]

**Key Vocabulary:** [3-5 highlighted words/phrases with brief Chinese translation]

**Grammar Highlights:** [1-2 notable structures used]

---

## Task

Generate Band 7-8 answers for the following IELTS Speaking topics, using my personal profile to make every answer unique and authentic:

[在此粘贴当季题库话题]
```

## 五、使用流程

1. **填写 Personal Profile** — 越详细越好，这是 AI 个性化答案的素材源
2. **获取当季题库** — 搜索「2026年1-4月雅思口语题库」，获取 Part 1/2/3 全部话题
3. **分批生成** — 每次给 AI 5-10 个话题，避免一次性太多导致质量下降
4. **人工微调** — 生成后通读一遍，替换掉你不会发音或觉得不自然的词
5. **录音练习** — 对照答案录音，回听自己，逐步脱稿
6. **重组素材** — 不要死背答案，而是提取「素材模块」（一段经历、一个观点），灵活组合到不同题目
